JS World

JavaScript World is a blog website with JavaScript tutorials. It Uses HTML, CSS, JavaScript and Wordpress. Wordpress installation is headless and is used to provide data through API

Features:
  • Responsive Web Design
  • Front Page Carousel with automatic rotation
  • Disable carousel rotation
  • List of Tutorials
  • Tutorial Details
  • Search Tutorials
  • Contact with saving the details on MySQL
  • Create Users
  • Update user information
  • Post/Delete Comments

Changes:
  • Added dates at tutorials page so the user can know if it is a new or an old tutorial
  • Added Sort functionality at tutorials page. The user can sort the posts by date asc/desc and title asc/desc. It is useful for the user for better browsing the tutorials
  • Added Filter functionality at the tutorials page on the already fetched results. Removes the posts that does not contain the keyword the user searched. Very good for the user to find a tutorial that contains a specific keyword

Community Science Museum

Community Science Museum is a science museum website. It uses HTML, CSS and JS only for contact form validation.

Features:
  • Responsive Web Design
  • Contact form validation
  • Contact form saves the messages to MySQL database through WP API

Changes:
  • Fixed readability problems by darkening the background color and increasing the font size. Text was difficult to read and somehow was creating problem to the eyes after a while.
  • Left aligned the paragraphs to help scalability and each line starts from the same axis. This helps the users eyes following the text.
  • Added JavaScript validation to the contact form. This feature was added to give immediately feedback to the user when he/she enters the information needed. Makes his experience better and avoiding entering information that is not needed. The feedback is shown at the bottom of the screen like an Android Toast message, and disappears after a while. It has different background colors from the site, so it will attract the users attention and notice the messages. Red for error and green for send confirmation.
  • Contact form saves the messages to database through WP API. This features is useful for the site administrators/owners but also for the user to have a way to communicate fast.

Gamehub

GameHub is an online store with PC games. It uses HMTL, CSS, JS and Wordpress as a headless CMS. It is used serve the games through API and save the messages from the contact form to MySQL database.

Features:
  • Responsive Web Design
  • Headless CMS
  • Save messages to the Database
  • List of Games
  • Game Details
  • Search Games
  • Game Categories that are shown dynamically
  • Shopping Cart
  • Check out process
  • Account Management simulation with localstorage:
    • Register/Login user
    • Update Account Settings
    • Update User information
    • Order history

Changes:
  • System Requirements has a different background color so it looks that it does not belong to the main section and confuses the user.
  • At the contact form changed the user's feedback for better experience. Message with green text below the "SEND" button and clearing the form.
  • Toggles button state at the contact form according to the form validation with JS. This way the user will not try to press it while there are missing information in the form. The button also changes colors while disabled for better user feedback.
  • Saves the messages from the contact form to the database for faster user communication with the site's administrators.
  • Fixed the error at the HTML that it was showing [object object]. Not good to have errors while the website is live.
